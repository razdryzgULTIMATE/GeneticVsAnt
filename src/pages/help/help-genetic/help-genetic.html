Здесь описано зачем нужен каждый параметр в генетическом алгоритме, что делать и куда нажимать!

"Картой" будем называть область, которая обведена в рамку

Перейдём к описанию параметров.

  1. Максимальное количество городов - нужно для генерации городов (см. описание кнопки "Сгенерировать города"). Должно быть задано > 1 города.
  2. Максимальное число поколений - этот параметр отвечает за то, сколько раз будет повторяться алгоритм. Значение должно быть > 0.
  3. Максимальная популяция - параметр отвечает за то, сколько особей будет в каждом поколении. Значение должно быть > 0.
  4. Вероятность мутации - параметр отвечает за то, как часто будут мутировать особи. Значение должно быть в диапазоне [0;1].
  5. Коэффициент отбора существ - параметр отвечает за то, сколько особей останется при отборе. Значение должно быть в диапазоне [0;1].

Перейдём к описанию кнопок.

  1. Сгенерировать города - кнопка "Сгенерировать города" отвечает за создание городов в случайных точках "карты". Для корректной работы нужно указать максимальное количество городов.
  2. Заполнить тестовыми данными - кнопка создаёт города из заранее заготовленных данных. Ничего дополнительно указывать не надо.
  3. Начать поиск - кнопка "Начать поиск" запускает работу алгоритма. По завершении алгоритм нарисует красной линией минимальный найденный путь(но это приближённое решение!). Для корректной работы нужно заполнить все поля, кроме максимального количества городов(на "карте" должно быть 2 города и больше).

Отмечу также, что пользователь может сам расставить города в любом порядке. Для этого нужно нажать ЛКМ в области "карты".
